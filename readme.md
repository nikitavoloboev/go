## `flow` CLI

[flow](https://github.com/nikitavoloboev/flow) repo explains purpose of flow well. See [cli/flow](cli/flow) for its code/docs.

Run `task install-flow` to install `fgo` (shortened `flow-go`) binary into `~/bin`.
